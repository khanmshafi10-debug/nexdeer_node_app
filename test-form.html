Inspect Fluent Form Fields

Go to: https://go.nexdeer.com/?fluentform=1

Then:

  1. Right-click on the Email field → Inspect Element
  2. Look for the name attribute in the <input> tag
  3. Right-click on the Message field → Inspect Element
  4. Look for the name attribute in the <textarea> tag

The name attributes you find are the exact field names we need to use!

Common Fluent Forms Patterns:

  • Email: email, input_email, email_1
  • Message: message, description, input_textarea, message_1
